# angular_learning_ek
Learning angular<ba>
How to run project:<br>
1. Open by VScode and install "Angular extension pack" together with "Angular cli"<br>
2. run command to create localhost<br>
```nb s``` <br>
3. Open in Chrome browser<br>
```http://localhost:4200/```<br>
